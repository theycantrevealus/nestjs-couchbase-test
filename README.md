Testing purpose only for [this library](https://github.com/theycantrevealus/nestjs-couchbase)

1. Add .env

```shell
APP_PORT=3000
COUCHBASE_CONNECTION_STRING=couchbase://localhost
COUCHBASE_BUCKET=bucket
COUCHBASE_USERNAME=user
COUCHBASE_PASSWORD=password

```

2. Start test

```shell
nest start gateway
```
